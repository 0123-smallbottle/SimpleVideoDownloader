# SimpleVideoDownloader
#How to use:
run "cli.bat"
put the link on it
wait for it
Notice:
if it is finished, you'll see a message box.
Press 'ok' button to close the script
(The video will in "video" folder)

'yt-dlp' file is from:
https://github.com/yt-dlp/yt-dlp
'ffmpg' file is from:
https://ffmpeg.org/
